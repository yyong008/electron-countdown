# electron countdown

a desktop count down can help you mananger your time。

## skills

- 🃏electron
- 🎇react
- 🌈tailwindcss
- 🚀vite

## before

```sh
# .npmrc
ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/"
```

## usage

```sh
git clone https://github.com/yyong008/electron-countdown.git

pnpm install

pnpm run dev # in dev

pnpm run build # build

# set tag
pnpm tag v0.0.2

# push tag
pnpm push:tag
```
